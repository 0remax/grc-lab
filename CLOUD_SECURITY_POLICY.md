# Cloud Security Policy
This policy mandates the security standards for all cloud infrastructure.

## Key Requirements
1. **MFA:** Mandatory for all users and administrative accounts.
2. **Patching:** Critical vulnerabilities must be patched within **24 hours**.
3. **Encryption:** All data at rest must be encrypted using AES-256.
